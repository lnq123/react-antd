


# Run Backend

```Bash
    1.cd backend/
    2.npm run rebuild-prod
    3.nohup npm run start-prod &
```


# Run Frontend

```Bash
    1.cd front/
    2.yarn install
    3.yarn start
```


# Running

DEV:
  - BUILD: npm run rebuild-dev
  - RUN: npm start
  
PROD:
  - BUILD: npm run rebuild-prod
  - RUN: npm run start-prod
